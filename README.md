# README

## Glavne funkcionalnosti aplikacije

1. **Uporabniška prijava in registracija** – omogoča ustvarjanje uporabniških računov ter varno prijavo v sistem.
2. **Urejanje uporabniškega profila** – uporabniki lahko posodabljajo osebne podatke, sliko profila in nastavitve.
3. **Dodajanje in upravljanje vsebin** – omogoča ustvarjanje, urejanje in brisanje vsebin znotraj aplikacije.
4. **Iskalnik** – omogoča hitro iskanje podatkov po različnih kategorijah.
5. **Obvestila v realnem času** – uporabniki prejmejo sprotna obvestila o pomembnih dogodkih ali posodobitvah.
